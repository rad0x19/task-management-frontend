## Installation
```sh
# use nodejs v18.17.0 or newer
yarn install 
# or
npm install
```
- update nuxt.config.js line 34 & 40, change the current value or target key to api domain you want to use. without "/".


## Start in dev mode
```sh
yarn dev
# or
npm run dev
```
